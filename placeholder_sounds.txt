Sound File Placeholders

This project requires the following sound files to be added manually:

1. battle.mp3      - Sound effect for land battles
2. naval_battle.mp3 - Sound effect for naval battles
3. march.mp3       - Sound effect for troop movements
4. speech.mp3      - Optional background sound during narration

These files should be placed in the root directory of the project. You can find suitable free sound effects from websites like:

- https://freesound.org/
- https://www.zapsplat.com/
- https://mixkit.co/free-sound-effects/
- https://soundbible.com/

Please ensure you have the rights to use any sound files added to the project.
